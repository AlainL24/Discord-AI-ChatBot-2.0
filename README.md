Cloud AI Discord Bot v2.0

This is version 2.0 of my AI Chatbot, a research-focused AI Discord bot that I made for my personal server.

## Features (v2.0 vs v1.0)

- Fully functional `/ask` slash command
- Research-based answers using DuckDuckGo + Llama3
- Filters out dictionary, grammar, and irrelevant sources
- Shows question and answer separately
- "🔎 Researching your question..." message while processing
- Proper guild command syncing
- Secure `.env` configuration
- Clean GitHub-ready structure

## Requirements

- Python 3.10+
- Discord bot token
- Ollama Llama3 model

## Installation

1. Clone the repo
2. `pip install -r requirements.txt`
3. Create `.env` with your Discord token, guild ID, and Ollama model
4. Run: `python bot.py`

## Usage

Type `/ask` in your server followed by your question.
